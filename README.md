# P2HACKS2025 アピールシート

## プロダクト名
favo (ふぁぼ)

## コンセプト
キラキラ×通知から生まれました。「通知が届く、その瞬間を宝物に」をテーマとして開発しました。日常に溢れる無機質な通知を、特別な人とのキラキラした瞬間に変える。メッセージが届くたびに心が弾むそんなアプリをめざしました。

## 対象ユーザ
ターゲット層
大切な人からの通知を、誰よりも特別な形で受け取りたい人 ex(好きな人、推し、家族など)

既存の通知に飽きた人達。コミュニケーションに新鮮なワクワクを求めている人。

## 利用の流れ
事前に自分のアカウントで通知を変えたい人のLINEかXアカウントをフォローする必要があります。
1．追加したい相手のLINEの名前またはXの名前を入力する。
2．通知の音、フラッシュを点灯するか決める。
3．保存するとホーム画面に相手が追加される。
4．相手がLINE送ったり、Xを投稿すると特別な通知が流れる。

## 推しポイント
作ってみてびっくりしたんですが、通知の速度がLINEとXよりもすごく早かったです！ここはだけは胸を張れます！
通知を変えることができるため、特別感が出るところです！興味ない人の通知を無視できるようになります！
